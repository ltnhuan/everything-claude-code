# EraGenAI (ECC) — Tài liệu tiếng Việt

Trang này là điểm vào tiếng Việt để bạn đọc nhanh, onboard và tiếp tục phát triển trên repo.

## Tài liệu nên đọc theo thứ tự

1. **Tổng quan bộ khung + cách dùng Claude/GPT**  
   [`FRAMEWORK-OVERVIEW.md`](./FRAMEWORK-OVERVIEW.md)
2. **README chính (bản tiếng Anh, cập nhật đầy đủ nhất)**  
   [`../../README.md`](../../README.md)

## Nội dung chính trong tài liệu tiếng Việt

- Quy mô hệ thống: **47 agents, 181 skills, 79 commands, hooks tự động**
- Cách chạy workflow thực tế theo tình huống:
  - làm tính năng mới
  - xử lý build fail / bug production
  - scale cho team
- Hướng dẫn code theo chu trình chuẩn: **Plan → TDD → Review → Security**
- Mẫu prompt và starter pack để team áp dụng ngay
- Chuẩn hóa đa ngôn ngữ để duy trì docs song song giữa tiếng Việt và tiếng Anh

## Gợi ý cho developer mới

- Bắt đầu từ `FRAMEWORK-OVERVIEW.md`, chạy theo checklist cuối tài liệu.
- Dùng commit nhỏ, có thể rollback, và luôn kèm test plan.
- Nếu tài liệu tiếng Việt chưa có phần bạn cần, đối chiếu với README tiếng Anh rồi cập nhật bổ sung theo chuẩn đa ngôn ngữ.
